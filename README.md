# spring-batch-learn
Spring Batch 4.3 Sample Project

### Getting Started

1. Git Clone
```
$ git clone https://github.com/munig629/spring-batch-learn.git
```

2. Packaging with maven wrapper
```
$ ./mvnw package
```

### Environment
- Java 11
- Maven
- Spring Boot 2.7.14
- Spring Batch 4.3.8
- MyBatis Framework 3.5.13
- H2 Database
- Lombok
